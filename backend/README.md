# AI Chat Backend

Secure Express.js backend for AI Chat platform.

## Quick Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```

## Environment Variables

Create `.env` file:
```
PORT=3001
NODE_ENV=development
FRONTEND_URL=http://localhost:5173
```

## API Endpoints

### POST `/api/chat/message`
Send a chat message to the AI.

### GET `/health`
Health check endpoint.
